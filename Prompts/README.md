# Prompts

This directory contains the prompt outlines used in our study.

## Contents

### PerGent Agent Prompts

Prompt templates powering the agentic persona generation pipeline (**PerGent**):

- `Prompt_PerGent_Generator_Agent.pdf` — Prompt outline for the PerGent Generator Agent responsible for creating persona items.
- `Prompt_PerGent_Critic_Agent.pdf` — Prompt outline for the PerGent Critic Agent responsible for evaluating and providing feedback on generated persona items.

### Baseline Prompts

Prompt templates for the baseline approaches evaluated in the paper:

- `Prompt_OneShot.pdf` — Prompt outline for the OneShot baseline approach which generates persona items in a one-shot (non-agentic) manner with no external resources.
- `Prompt_OneShot+Res.pdf` - Prompt outline for the OneShot+Res baseline approach, which includes external resources for persona generation.
